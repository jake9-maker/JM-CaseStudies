<task type="auto">
  <name>Task 3: Configure Hugo config.toml and establish build output</name>
  <files>config.toml</files>
  <action>
    1. Edit config.toml to configure:
       - baseURL: "https://jmaisel.github.io/JM-CaseStudies/" (GitHub Pages URL for repo)
       - title: "JM Case Studies"
       - languageCode: "en-us"
       - [markup.goldmark] to enable unsafe HTML if needed: unsafe = true
       - [outputs] to generate HTML (no JSON/RSS for v1)
       - [taxonomies] to add "pillar" taxonomy for the three pillars (complex-problem-solving, ai-integration, ux-research)
    2. Verify build outputs to public/ directory (Hugo default)
    3. Run `hugo` to generate static site into public/
    4. Verify .gitignore excludes /public/ and /resources/
    Rationale: baseURL is critical for GitHub Pages - must be exact repo URL. Taxonomies enable filtering/organizing by pillar later. Goldmark unsafe HTML allows for flexibility in markdown.
    Avoid: Don't set minify: true yet (Phase 4 handles optimization). Don't enable caching headers (Phase 4 handles deployment config).
  </action>
  <verify>
